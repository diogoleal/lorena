Lorena

Project Lorena is a Mame frontend.

Welcome to Project Lorena.

This is a simple and direct frontend to run Mame (mamedev.org) in Linux, Windows and Mac OS X. Where the user has quick access to games information, including pictures, videos, sounds, etc.
